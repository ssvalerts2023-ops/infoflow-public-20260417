### 1) Presort check (MANDATORY FIRST STEP)

* DATABASE

This includes pages that look like: “Shipment records / Importer / Exporter / HS code / Bills of lading / manifests / trade data”.

* LIST
  `target_company` appears only in a generic list with no transaction evidence (examples: competitor list, vendor list without deal details, exhibitor/attendee list, directory listing, forum user list, “top companies” list, “clients” list with no contract/shipment/invoice signals).

* RELEVANT
  Not DATABASE and not LIST, but the file contains at least one concrete signal of a commercial relationship involving `target_company` and at least one counterparty, OR contains weaker/indirect but still transaction-linked references (e.g., “invoice issued to…”, “supplier to…”, “shipped to…”, “paid by…”).  

* NOT_RELEVANT
  None of the above.

* `presort.category` = "DATABASE" | "LIST" | "RELEVANT" | "NOT_RELEVANT"
